# BudgetHub

A private, offline-first budget management app. All data stays on your device.

## Features

### Fixed Expenses
- Add, edit, and delete expense categories
- Single-payment or multi-entry (tracked) types
- Editable amounts per month with paid checkmarks
- Monthly payment history

### Subscriptions
- Track active and cancelled subscriptions
- Billing day reminders (opt-in browser notifications)
- Calendar integration (.ics export, 3 days before billing)
- Monthly planning notes with checkboxes

### Health & Outing
- Separate budgets with progress tracking
- Unused amounts auto-shift to savings
- Expense logging with notes

### Savings
- Combined view across all budget sources
- Month-by-month breakdown
- All-time savings total

## Privacy
- **No server, no database, no tracking**
- All data stored in browser localStorage
- Source code contains zero personal information
- Safe for public GitHub hosting

## Setup
1. Upload `index.html` to a GitHub repository
2. Enable GitHub Pages (Settings → Pages → Deploy from branch)
3. Open the URL on your phone
4. Add to Home Screen — works like a native app

## Export
One-tap CSV export of all data for use in Excel or Google Sheets.
